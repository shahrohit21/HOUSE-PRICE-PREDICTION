# House Price Prediction using Machine Learning

## Problem Statement
The goal of this project is to predict house prices based on various features such as area, number of bedrooms, bathrooms, and other property details using Machine Learning techniques.

---

## Project Overview
This project uses multiple regression models to predict house prices and compares their performance. The final model is deployed using a Streamlit web application for real-time predictions.

---

## Models Used
- Gradient Boosting Regressor (GBR)
- Random Forest Regressor (RF)
- XGBoost Regressor (XGB)

---

## Model Performance
| Model | R² Score |
|------|---------|
| Gradient Boosting | 0.87 |
| Random Forest | 0.84 |
| XGBoost | 0.85 |

Final model selected: **Gradient Boosting / Ensemble Model**
